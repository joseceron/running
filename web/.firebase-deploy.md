# Deploy a Firebase Hosting

## Setup inicial (una vez)

```bash
npm install -g firebase-tools
firebase login
# crear proyecto en https://console.firebase.google.com
# id sugerido: academia-ia-mvp
firebase use --add  # seleccionar el proyecto creado
```

## Build + Deploy

```bash
cd landing
npm run build        # genera /out con HTML estático
firebase deploy --only hosting
```

## Dominio custom

Después de `firebase deploy`, ir a Firebase Console → Hosting → "Add custom domain":
- academia.ai (o el dominio final)
- Firebase genera certificado SSL gratis (Let's Encrypt) y resuelve DNS

## CI/CD opcional

Si quieres deploys automáticos desde GitHub:

```bash
firebase init hosting:github
```

Crea workflow `.github/workflows/firebase-hosting-merge.yml` que despliega a producción al hacer merge a `main`.
